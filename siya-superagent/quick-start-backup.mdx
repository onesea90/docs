## Welcome to Siya!

This guide will help you get started with Siya in just a few minutes. Let's begin your journey with the most advanced AI assistant available today.

## System Requirements

### macOS Desktop App
- **Operating System**: macOS 11.0 (Big Sur) or later
- **Memory**: 8GB RAM minimum (16GB recommended for best performance)
- **Storage**: 2GB available disk space
- **Internet**: Required for AI features
- **Processor**: Apple Silicon (M1/M2/M3) or Intel processor

## Installation

### Step 1: Download Siya
1. **Download Siya for macOS**:
   - [Download for Apple Silicon (M1/M2/M3)](https://siya.com/join-waitlist.html)
   - Intel version coming soon
2. The Siya.dmg file will download to your Downloads folder

### Step 2: Install the Application
1. Open the downloaded Siya.dmg file
2. Drag the Siya icon to your Applications folder
3. Eject the disk image

### Step 3: First Launch
1. Open Siya from your Applications folder
2. If prompted, click "Open" to confirm you want to run the app
3. Grant necessary permissions when requested

## Initial Setup

### Account Setup
1. Create your Siya account or sign in
2. Choose your subscription plan
3. Complete the onboarding process

### Preferences
Navigate to Siya → Preferences to customize:
- **Theme**: Choose between light and dark mode
- **Shortcuts**: Set up keyboard shortcuts
- **Notifications**: Configure alert preferences

## Understanding the Interface

### Main Window
- **Input Area**: Where you type or paste your requests
- **Conversation View**: Shows the ongoing dialogue with Siya
- **Status Bar**: Displays current activity and agent status

### Toolbar
- **New Chat**: Start a fresh conversation
- **History**: Access previous conversations
- **Settings**: Quick access to preferences
- **Help**: Get assistance and documentation

## Execution Modes

### Chat Mode (Interactive)
Perfect for:
- Exploratory questions
- Step-by-step guidance
- Iterative refinement
- Learning new concepts
- Direct MCP tool usage

To use: Simply start typing your question or request

### Task Mode (Autonomous)
Ideal for:
- Complex projects
- Multi-step operations
- Batch processing
- Automated workflows
- Production tasks

To use: Start your request with "Create", "Build", "Analyze", or similar action words

## Working with Files

### Uploading Files
1. Drag and drop files into the chat window
2. Or click the attachment button
3. Supported formats include:
   - Text files (.txt, .md, .json)
   - Code files (.py, .js, .java, etc.)
   - Data files (.csv, .xlsx)
   - Documents (.pdf, .docx)

### Saving Results
- Click the save button on any response
- Use keyboard shortcut ⌘+S
- Results are saved to your designated folder

## Extending Siya

**Examples**:

## Best Practices

### Writing Effective Prompts
1. **Be Specific**
   - ❌ "Help with Python"
   - ✅ "Create a Python script that reads a CSV file and generates a bar chart"

2. **Provide Context**
   - ❌ "Fix this error"
   - ✅ "I'm getting a TypeError in my React component when trying to map over an array"

3. **Include Examples**
   - ❌ "Format this data"
   - ✅ "Format this data like: Name | Age | City"

### Getting Better Results
- **Break Down Complex Tasks**: Let Siya create a plan first
- **Use Follow-up Questions**: Refine results through conversation
- **Provide Sample Data**: Include examples of input/output
- **Specify Constraints**: Mention any limitations or requirements

## Siya for macOS

Experience the full power of Siya with our native macOS application. Designed specifically for Mac users, it combines powerful AI capabilities with the elegant interface you expect.

## System Requirements

### Minimum Requirements
- **macOS**: 11.0 (Big Sur) or later
- **Processor**: Intel or Apple Silicon (M1/M2/M3)
- **Memory**: 8GB RAM
- **Storage**: 2GB available space
- **Internet**: Required for AI features

### Recommended Specifications
- **macOS**: Latest version
- **Processor**: Apple Silicon for best performance
- **Memory**: 16GB RAM or more
- **Storage**: 5GB+ for workspace
- **Internet**: High-speed connection

## Installation

### Download & Install
1. **Download** Siya for macOS:
   - [Apple Silicon (M1/M2/M3)](https://siya.com/join-waitlist.html)
   - Intel version coming soon
2. **Open** the downloaded .dmg file
3. **Drag** Siya to Applications folder
4. **Launch** from Applications or Launchpad

### First Launch
- **Security**: Click "Open" if macOS asks
- **Permissions**: Grant necessary access
- **Sign In**: Use your Siya account
- **Setup**: Complete quick onboarding

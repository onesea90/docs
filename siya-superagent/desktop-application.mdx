        - Close unused windows
      </Step>

        - Check internet connection
        - Switch to faster model
        - Disable unnecessary features
      </Step>
      
      <Step title="Crashes">
        - Check crash logs in Console
        - Reset settings to default
        - Reinstall if persistent
      </Step>

      
      ```bash Diagnostic Commands
      # Check Siya version
      ```

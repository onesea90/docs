```mermaid
sequenceDiagram
    participant User
    participant Main as Main SIYA Agent
    participant SWE as SWE Agent
    participant Search as Search Agent
    participant Browser as Browser Agent
    participant Memory as Shared Memory
    
    User->>Main: Complex request
    Main->>Main: Analyze & Plan
    
    par Parallel Execution
        Main->>Search: Research requirements
        and
        Main->>SWE: Prepare environment
    end
    
    Search->>Memory: Store findings
    SWE->>Memory: Update progress
    
    Main->>Browser: Execute web tasks
    Browser->>Memory: Save results
    
    Main->>Memory: Aggregate all data
    Memory-->>Main: Combined results
    Main-->>User: Final deliverable
    
    Note over Main,Memory: Continuous synchronization
```
</Frame>
